# 📊 Smart Expense & Budget Tracker

A highly optimized, professional full-stack web application designed to help users track personal finances and monitor monthly spending constraints in real time. Built from scratch with a robust Python backend and a responsive, modular frontend interface.

## 🚀 Technical Architecture Stack

- **Backend Server Framework:** Python Flask (RESTful JSON API Routing Architecture)
- **Database Management System:** SQLite3 (Structured relational storage with transactional query mapping)
- **Frontend Engine Layers:** Semantic HTML5 / Clean Modular CSS3 Layout Framework
- **Client-Side Scripting Nervous System:** Asynchronous Vanilla JavaScript (Asynchronous Fetch API / Real-Time DOM Manipulation Architecture)

## 💎 Key Operational Engineering Features

1. **Dynamic Visual Progress Bar:** Calculates real-time expenditure against a fixed configuration ceiling limit parameter (`₹10,000`). Utilizes CSS 0.4s smooth animations and implements an automatic structural threshold alert that flashes warning red (`#dc3545`) if spending hits 80% or higher.
2. **Native Browser Localization API Integration:** Dynamically parses raw integer data columns and formats text displays instantly to follow the official Indian Numbering System standards (`en-IN`) with proper comma divisions and trailing decimal zeros across both top summary banners and table row units.
3. **Optimized DRY Architecture & DOM Cell Factory:** Implements decoupled design structures by separating application logic into localized sub-modules. Uses a central custom helper factory line to eliminate repetitive layout code clutter inside rendering loops.
4. **Full CRUD Deletion Capabilities:** Features complete item log tracking drop operations via dedicated database rows linked straight to unique tracking IDs via interactive buttons.

## 🛠️ How to Launch the Application Locally

To spin up this environment on your machine, clone this repository and execute these commands inside your terminal:

```bash/powershell
# 1. Start the Flask local deployment application server engine loop
python app.py
```

Then, open your web browser tab canvas and navigate directly to:
👉 `http://127.0.0.1:5000`
